## üîç Quick Reference

### Common Tasks

**Install Dependencies**
```bash
npm install
```

**Set Up Database**
```bash
npx prisma db push
npx prisma generate
```

**Run Development Server**
```bash
npm run dev
```

**Build for Production**
```bash
npm run build
```

**Deploy to Vercel**
```bash
vercel --prod
```
